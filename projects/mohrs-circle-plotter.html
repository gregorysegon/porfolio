Mohr’s Circle Plotter

Problem

Materials testing often requires Mohr’s Circle to find principal stresses/strains. Strain rosettes measure strain; stresses are derived using Hooke’s Law.

Solution

Built a MATLAB app to generate Mohr’s Circle for strains, plane stress, and 3-D stresses, including transformations for a specified angle.

Tools Used

MATLAB App Designer

Highlights

  • Supports strain and stress circles
  • Angle-based transformations built in
