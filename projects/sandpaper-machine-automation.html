Sandpaper Machine Automation

Problem

Sandpaper Machine wanted to automate loading, fixturing, and unloading 6"×6" CNC-textured wood blocks. Blocks had inconsistent geometry and groove locations due to manual cutting.

Solution

Designed a fixture with two fixed edges and a 45° pushing corner to self-align blocks despite variation. Added a ramp + lip feature to support the block during milling and designed a gripper that uses the bottom groove while clearing fixture edges.

Tools Used

SOLIDWORKS3D printerMill (for some purchased parts)KUKA KR-6 (test setup)

Highlights

  • Self-aligning fixture reduces placement precision required
  • Accommodates block size variation via corner travel
  • Gripper/fixture share common groove interface
