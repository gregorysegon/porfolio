Combined Tool (SIP List, Order Comments, Daily Cancelled List)

The Call Center uses three reports (SIP list, Order Comments, and Daily Cancelled list) generated up to three times a day. Staff spent ~15–40 minutes per report removing unnecessary data and formatting it into something usable.

Order Counting Automation

Every two weeks, Epic exports orders placed by each user. Tracking totals required filtering, totaling, and emailing each employee—a time-consuming manual process.

BC TAT Report Automation

Lab staff used a CSV file generated by Virtuo incubators to track positive blood culture TAT. The process was manual: reformatting, then individual Sunquest lookups for call times, then manager-side graphing.

Sunquest Handling Counter

The lab proccessing staff productivity is caculated using multiple reports that are combined to track how many activies each tech preformed in Sunquest. This process took ~4 hours to complete.
