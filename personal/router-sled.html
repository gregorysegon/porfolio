Router Sled

Problem

After gluing cutting boards, the surface can warp and no longer be flat. A router sled keeps a router moving in a flat plane to surface the board.

Solution

Designed an adjustable router sled using 2020 aluminum extrusions and wheeled side pieces that ride the table edges. Included router depth adjustment via a sliding mount and set screw.

Tools Used

SOLIDWORKS3D printer2020 aluminum extrusionsMiter sawTaps/fasteners

Highlights

  • Adjusts to multiple tabletop widths
  • Depth adjustment via sliding router mount
  • Used to surface end-grain and brick-pattern boards
