Remote System for Mini Split AC Units

Problem

Five indoor evaporators share one condenser and must be set to similar climate settings. Adjusting each unit room-by-room with an IR remote is inconvenient, and the manufacturer's add-on modules were too costly for all five units.

Solution

Built a centralized control system using a Raspberry Pi UI that sends commands to each room module. Each room module (ESP-8266 + DHT11 + IR transmitter) reports temperature/humidity back to the Raspberry Pi via MQTT and receives IR commands to control the evaporator.

Tools Used

Raspberry PiNode-REDMQTTESP-8266DHT11 sensorIR transmitterAnalysIR

Highlights

  • Single interface to control multiple evaporators
  • Room-by-room temp/humidity feedback to dashboard
  • Reverse-engineered IR commands for compatibility
