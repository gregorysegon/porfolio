Parking Distance Sensor

Problem

Inconsistent parking position in the garage made it hard to move around the front bumper and open the hatch with the garage door closed.

Solution

Built an ultrasonic-based distance indicator. LED colors change to tell the driver to keep pulling in, stop, or back up based on distance thresholds.

Tools Used

Arduino UnoUltrasonic sensorLEDsSOLIDWORKS3D printer

Highlights

  • Visual distance feedback with LED states by range
  • Custom 3D-printed enclosure
